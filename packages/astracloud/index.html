FAQ

Questions? Answered.

Can I self-host AstraOS instead?
Absolutely! AstraOS is fully open-source (MIT). Run npx astra-os or bash setup.sh to set up on your own server. AstraCloud is for those who want zero DevOps.
Do I need my own LLM API key?
Yes. AstraOS connects to Claude, GPT-4o, Gemini, or Ollama. You provide your own API key and pay the LLM provider directly — full control over costs and model selection.
What's included in every plan?
All 55+ skills, 14+ channels, admin dashboard, workflow builder, marketplace, auto-updates, daily backups, and enterprise security. Higher plans add more compute + premium features like SSO and audit log.
Can I migrate from MyClaw / OpenClaw?
Yes. AstraOS includes a built-in migration tool that imports your skills, settings, and data from OpenClaw.
Is my data private?
100%. Each instance runs in isolation. Data encrypted at rest (AES-256-GCM) and in transit (TLS 1.3). We never access your data. Enterprise plans include data residency for 7 regions.
Can I cancel anytime?
Yes. No contracts, no lock-in. Cancel and export your data anytime. 7-day money-back guarantee.
