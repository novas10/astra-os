Security-First Architecture

Three Layers of Protection

AstraOS v3.5 addresses every known vulnerability in competing platforms with dedicated security subsystems. No plaintext credentials. No token leaks. No unsigned skills.

GatewayShield
Gateway Security Layer

Comprehensive request-level protection that prevents CVE-2026-25253-class attacks, CSRF exploits, and brute force attempts before they reach your agents.

  • Token security — never accepts auth tokens from URL query params
  • CSRF protection — double-submit cookie with SameSite=Strict
  • Brute force lockout — 10 attempts / 30-minute lockout window
  • Public exposure detection with alerting
  • Security headers — HSTS, X-Frame-Options, CSP
  • IP allowlist / denylist network access control
  • Security scoring and grading (A+ to F)
CredentialVault
Encrypted Secret Storage

All API keys, tokens, and secrets encrypted at rest using AES-256-GCM. Never stores credentials in plaintext. Solves the plaintext credential vulnerability found in competing platforms.

  • AES-256-GCM encryption with per-credential IV and salt
  • GCM authentication tags for tamper detection
  • Key rotation with zero-downtime re-encryption
  • Access audit trail for every credential operation
  • Auto-expiry for time-limited credentials
  • Encrypted import / export transport
SkillSandbox
Skill Security & Signing

Deep static analysis and cryptographic signing prevents supply-chain attacks. Every skill is analyzed, signed, and sandboxed before activation.

  • Ed25519 cryptographic signing for verified publishers
  • Deep static analysis — AST-level pattern detection
  • Permission manifest enforcement
  • Reputation scoring tracking safety history
  • Auto-quarantine system for flagged skills
  • Community reporting for crowdsourced security

Voice AI

Talk Mode

Full-duplex conversational voice. Interrupt the agent mid-sentence. Activate by wake word or push-to-talk. Powered by ElevenLabs TTS and Whisper/Deepgram STT.

Talk Mode Active
listening → processing → speaking → interrupted
  • ~
    Interrupt Support
    Interrupt the agent mid-sentence. It stops speaking immediately and starts listening.
  • !
    Wake Word Detection
    Configurable wake word activation. Always-on listening with low-power keyword spotting.
  • #
    Push-to-Talk
    Manual activation mode for noisy environments. Hold to speak, release to process.
  • >
    ElevenLabs TTS
    High-quality text-to-speech with configurable voice, stability, and similarity boost.
  • <
    Whisper / Deepgram STT
    Speech-to-text with confidence scoring, language detection, and streaming transcription.
  • =
    Voice Activity Detection
    Configurable VAD sensitivity and silence timeout for natural conversation flow.
